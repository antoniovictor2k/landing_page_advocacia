# Landing Page - Escritório de Advocacia

Landing page profissional para escritório de advocacia desenvolvida em Flutter com integração ao WhatsApp.

## Recursos

- Design responsivo (Desktop e Mobile)
- Áreas de atuação
- Apresentação da equipe
- Depoimentos de clientes
- Integração com WhatsApp
- Mapa de localização
- Interface profissional e elegante

## Como executar

```bash
flutter pub get
flutter run -d chrome
```
